Web開発基礎 課題1

課題1 テーブル(HTML)を出すプログラムを作ってみる

要件

教科(国語、英語、数学、歴史、生物)の点数をテーブル形式で表示する。

元データは↓のように、連想配列で与えられる。

        $results = array(
            "Japanese" => 60,
            "English" => 70,
            "Mathematics" => 50,
            "History" => 60,
            "Biology" => 45
        );

出力は↓のように表(table)にしたい

Japanese60
English70
Mathematics50
History60
Biology45
Total285
  1. 教科の数はサンプルは5個だが、増減に対応可能にする。
  2. 合計点を計算して表示する。
  3. 60点未満の場合は、赤字にする。(赤字でなくても良いが、他と別の表示にしたい)

check1.phpに書いていき、できたら提出してください。

作成経過は、check1_answer1.php, check1_answer2.php, check1_answer3.php
