Web開発基礎 PHPとJSの文法の違い

P63: 変数

PHP

$ + アルファベット、数字、アンダースコア

※数字は先頭には使えない

※ $の後には空白ナシ

$name = '山田';

JavaScript

アルファベット、数字、アンダースコア　

※数字は先頭には使えない

※変数を最初に宣言するときは let + 空白 + 変数名、または const + 空白 + 変数名

let name = '山田';

P64: 結合演算子

PHP

文字列の場合 .

数字の場合 +

$a = 1;
$b = 2;
echo ($a . $b); // 12
echo ($a + $b); // 3

※もし文字列だった場合でも + でつなげると足し算する

$a = '1';
$b = '2';
echo ($a . $b); // 12
echo ($a + $b); // 3

JavaScript

文字列の場合も数字の場合も +

※変数が、文字列か、数かによって結果が異なる

PHP も JavaScript も、do ～ while

6章 P105 配列

PHP

配列は array(...) で作る or [...]で作る

$friends = array("はるき","かおる","ひでと");

↑と↓は同じ配列を作る

$friends = ["はるき","かおる","ひでと"];

最初の要素が$friends[0]となる

JavaScript

[...]で作る

let friends = ["はるき","かおる","ひでと"];

P111 連想配列

PHP

=> を使うことで、キーと値を設定する

$result1 = array(
    'Japanese' => 80,
    ...
);

JavaScript

JavaScriptでは、オブジェクトのプロパティが、PHPの連想配列に近い。

let result1 = {
    Japanese: 80,
    ...
}

P118 foreach

配列の要素ごとに繰り返し処理を行う構文

PHP

foreach $... as $... を使う

$fruits = array("りんご","みかん","バナナ");
foreach ($fruits as $fruit) {
    echo $fruit . "<br>";
}

foreach 元の配列 as 各々の要素

JavaScript

for ... of ...を使う

let fruits = ["りんご","みかん","バナナ"];
for (fruit of fruits) {
    console.log(fruit);
}

for 各々の要素 of 元の配列
